> 三极管 (Transistor) 基于 PN 结制造，通过**小电流控制大电流**，广泛应用于放大、开关和信号调节。

---

## 5.1 概述

三极管分为 **NPN 型**与 **PNP 型**。下文以 NPN 为主讲解，理解 NPN 后 PNP 自然类推。

---

## 5.2 分类

| 类型 | 结构 |
|------|------|
| **NPN** | 两个 N 区夹一个 P 区（可看作两个共用 P 区的二极管） |
| **PNP** | 两个 P 区夹一个 N 区 |

三个电极：**基极 (B)**、**集电极 (C)**、**发射极 (E)**。

---

## 5.3 基本结构

NPN 型可看成两个共用 P 区域的二极管。通过小电流（基极→发射极）控制大电流（集电极→发射极），实现**开关或放大**。

## 5.4 导通原理

### NPN 型

- **基极→发射极**电流较大 → 三极管**导通** → 电流从**集电极→发射极**
- 基极→发射极电流较小或无 → 三极管**关闭**

### PNP 型

- **发射极→基极**电流较大 → 三极管**导通** → 电流从**发射极→集电极**
- 发射极→基极电流较小或无 → 三极管**关闭**

---

## 5.5 NPN 与 PNP

| | NPN | PNP |
|---|-----|-----|
| **导通条件** | I_B→E 足够大 | I_E→B 足够大 |
| **电流方向** | C → E | E → C |
| **载流子** | 发射/收集的都是**电子** | 发射/收集的都是**空穴** |

### 嵌入式典型用法

```
NPN 开关（低端开关）：
  GPIO ──[Rb]── B
  E ── GND
  C ── 负载 ── VCC

PNP 开关（高端开关）：
  GPIO ──[Rb]── B
  E ── VCC
  C ── 负载 ── GND
```

## 5.6 快速分辨三极管

看电路符号：

1. **箭头方向**：始终由 **P 指向 N**
2. **中间引脚** = **B（基极）**
3. **箭头所在线** = **E（发射极）**
4. **剩下** = **C（集电极）**
