## 7. 蜂鸣器

发出声音的电子元器件，用于报警音、提示音、音频信号输出。

### 7.1 概述

按驱动方式分为：**有源蜂鸣器**、**无源蜂鸣器**。

> 这里的「源」指**音频振荡源**，不是电源。

### 7.2 分类

| | 有源蜂鸣器 | 无源蜂鸣器 |
|---|-----------|-----------|
| **别名** | 自激式 | 它激式 |
| **内部** | 含驱动/振荡电路 | 无振荡电路 |
| **使用** | 给外部电源即可，声调固定 | 需外部方波/PWM 驱动 |
| **外观** | 有贴纸，背部黑胶封闭，体积大 | 无贴纸，无黑胶，体积小 |
| **阻值** | 较大（含振荡电路） | 较小 |

### 7.3 关键参数

| 参数 | 说明 |
|------|------|
| **工作电压** | 正常发声电压，过高可能损坏 |
| **外观大小** | 需匹配 PCB 尺寸 |
| **驱动方式** | 确认有源还是无源 |
| **声压** | 声音强度，越高越响 |

> 嵌入式开发中，**无源蜂鸣器**需 MCU 输出 PWM 驱动；**有源蜂鸣器**GPIO 高低电平即可。

## 8. 电磁继电器

利用**电磁感应**原理的电磁开关，用**低电压、低电流**控制**高电压、高电流**电路。

### 工作原理

```
控制端：MCU GPIO → 小电流驱动线圈 → 产生磁场
被控端：触点闭合/断开 → 控制大电流负载（电机、220V 设备等）
```

### 关键参数

| 参数 | 说明 |
|------|------|
| **线圈电压** | 线圈工作电压，常见 5V、12V、24V |
| **最大切换电压** | 触点能安全切换的最大电压 |
| **最大切换电流** | 触点能安全切换的最大电流 |
| **动作时间** | 施加电压到触点完成接触的时间 |
| **触点形式** | 常开 (NO)、常闭 (NC)、转换 (COM) |

> 嵌入式常用 5V 继电器模块（含驱动三极管和续流二极管），直接用 GPIO 控制。
