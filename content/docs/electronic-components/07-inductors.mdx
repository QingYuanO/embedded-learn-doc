## 6.1 回顾物理定律

### 1. 安培定则

**定则一**：右手握通电直导线，大拇指指向电流方向，四指指向磁场方向。

**定则二**：右手握通电螺线管，四指指向电流方向，大拇指指向 N 极。

### 2. 法拉第电磁感应定律

闭合电路磁通量发生变化 → 闭合电路中产生感应电流。

### 3. 楞次定律

感应电流的磁场总要**阻碍**引起感应电流的磁通量变化。

---

## 6.2 概述

**电感器**把电能转换成磁能并存储，是「磁能的暂存器」。

**核心作用**：通过电能与磁能相互转化，平滑电流，对电流有「**来拒去留**」的反应。

- 电流突然出现或变大 → 电感抑制，平缓变化
- 电流突然消失或变小 → 电感抑制，平缓变化

## 6.3 单位与换算

| 项目 | 说明 |
|------|------|
| **单位** | 亨利 (H) |
| **1H 含义** | 电流变化率 1A/s 时，产生 1V 反向电压 |
| **换算** | 1H = 1000mH，1mH = 1000μH |

> 1H 也是很大的单位，实际常用 μH、mH 级。

**电感决定式**（了解）：

```
L = μ · N² · A / l
```

- μ：磁导率 (H/m)
- N：线圈匝数
- A：截面积 (m²)
- l：磁路长度 (m)

---

## 6.4 核心特点

| 特点 | 说明 |
|------|------|
| **电感储能** | 以磁场方式储存电能 |
| **隔交通直** | 对直流无阻碍，对交流阻碍大 |

与电容相反：**电容隔直通交，电感隔交通直**。

---

## 6.5 常见电感器

**磁环电感 / 贴片电感**：磁环作核心，绕制导线形成线圈。

**核心参数**：

| 参数 | 说明 |
|------|------|
| **电感值** | 存储磁能/抑制电流变化的能力 |
| **直流电阻 (DCR)** | 金属导体固有电阻，越小越好 |
| **精度** | 实际值与标称值误差 |
| **饱和电流** | 达到磁饱和的临界电流 |

## 6.6 电感的大小识别

### 数字 + 字母混合标注

1. **三位数字 + 一位字母**：前两位有效数字，第三位为倍乘数（加几个零）
2. **R / N 混合标注**（小功率贴片电感）：
   - R 表示小数点，单位 μH
   - N 表示小数点，单位 nH
