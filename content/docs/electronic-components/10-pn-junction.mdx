## 3. PN 结

### 3.1 概述

P 型半导体与 N 型半导体通过特殊工艺结合而成。

- P 型中**多子**是空穴
- N 型中**多子**是电子
- 二者都是电中性的

<Diagram src="/electronic-components/diagrams/pn-junction.svg" alt="PN结耗尽区" width={720} height={320} />

### 3.2 内建电场的形成

P 型与 N 型组合后：

1. N 区电子**扩散**到 P 区与空穴复合
2. N 区因缺电子带**正电**，P 区因得电子带**负电**
3. 内部形成**内建电场**，阻止更多电子从 N 扩散到 P
4. 中间形成**耗尽区**（空间电荷区 / 势垒区）

> 不仅电子会从 N 扩散到 P，P 区的空穴也会漂移向 N 区。空穴也是载流子。课程为便于分析，主要从电子扩散角度讲解，**记住 PN 结结论即可**。

### 3.3 PN 结的导通原理

#### 正向偏置（P 接正，N 接负）

外部电场使内建电场**变小** → 耗尽区缩小 → 继续加大外部电压可**抵消内建电场** → PN 结导通 → N 区电子处于自由状态 → **有电流通过**。

**结论：PN 结导通时，电流方向从 P 到 N。**

#### 反向偏置（P 接负，N 接正）

外部电场与内建电场**同向叠加** → 耗尽区**变大** → 几乎无电流（直到击穿）。

<Diagram src="/electronic-components/diagrams/pn-bias.svg" alt="PN结正反向偏置" width={720} height={260} />

```
正向：P ──(+)── 电源 ──(-)── N  →  导通
反向：P ──(-)── 电源 ──(+)── N  →  截止（耗尽区扩大）
```

---

## 核心结论

| 状态 | 条件 | 结果 |
|------|------|------|
| **导通** | P 正 N 负，电压 > 门槛 | 电流 P → N |
| **截止** | P 负 N 正 | 耗尽区扩大，几乎无电流 |
| **击穿** | 反向电压过大 | 反向电流急剧增大（稳压二极管利用此特性） |
