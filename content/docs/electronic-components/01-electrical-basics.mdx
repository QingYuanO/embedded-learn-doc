## 1. 前序知识

原子是构成物质的最小单位，由**原子核**（质子 + 中子）和**电子**组成。

<Diagram src="/electronic-components/diagrams/atom-structure.svg" alt="碳原子结构示意图" width={720} height={320} />

### 1.1 质子（Proton）

- 质子带 **1 个单位正电荷**
- **质子数量决定元素身份**——改变质子数就改变了原子类型

> 举例：若能从铅原子核移出 3 个质子，理论上可「炼」出金原子。实际上质子结合极其紧密，这也是炼金术未能实现的原因。现代核反应堆、粒子加速器虽可实现类似过程，但所需能量和成本远超黄金价值。

### 1.2 中子（Neutron）

- 中子**不带电**
- 对电学影响远小于质子，通常不深入探讨（不直接影响电势、电流形成）

### 1.3 电子（Electron）

1. 电子带 **1 个单位负电荷**，围绕原子核运动
2. 通常质子数 = 电子数，正负抵消，原子呈**电中性**
3. **电子层**：电子围绕原子核活动的范围

每层最多容纳 **2n²** 个电子（n 为层编号）：

| 层 | 公式 | 最多电子数 |
|----|------|-----------|
| 第 1 层 (n=1) | 2×1² | 2 |
| 第 2 层 (n=2) | 2×2² | 8 |
| 第 3 层 (n=3) | 2×3² | 18 |

> 最外层 8 个电子通常是较稳定状态（**八隅规则**）。氢 (H)、氦 (He) 只需 2 个电子即稳定。

**库仑定律**：异种电荷相吸，同种电荷相斥。

```
F = k · |q₁ · q₂| / r²
```

- q₁、q₂：两个电荷的电荷量
- r：两电荷间距离
- k：库仑常数（在不同介质中略有变化）

**最外层电子**更容易脱离原子，因距原子核较远、受引力较弱。

### 1.4「石墨」与「金刚石」

| 物质 | 结构 | 导电性 |
|------|------|--------|
| **石墨** | 每个 C 与 3 个 C 成共价键，外层剩 1 个自由电子可在层间移动 | **导体** |
| **金刚石** | 每个 C 与 4 个 C 成共价键，立体网状结构，无自由电子 | **绝缘体** |

<Diagram src="/electronic-components/diagrams/graphite-diamond.svg" alt="石墨与金刚石结构对比" width={720} height={280} />

> 石墨层间靠范德华力结合，层间易滑动，可作润滑剂和铅笔芯。

## 2. 电流

**概念**：电荷的**定向移动**形成电流。衡量大小 = 单位时间内通过导体横截面的电荷量。

**注意**：电学定义的「电流方向」是**正电荷**流动方向（19 世纪初定义）。后来发现是电子在移动，但为保持统一，**电流方向与电子移动方向相反**。

<Diagram src="/electronic-components/diagrams/current-direction.svg" alt="电流方向与电子移动方向" width={720} height={220} />

| 项目 | 说明 |
|------|------|
| **单位** | 安培 (A) |
| **1A 含义** | 1 秒内有 6.242×10¹⁸ 个单位电荷通过横截面 |
| **换算** | 1A = 1000mA |

> 6.242×10¹⁸ 个单位电荷的电荷总量 = **1 库仑 (C)**。  
> 因此 1A 也可描述为：1 秒内 1 库仑电荷通过横截面。

## 3. 电场、电势、电势能

为理解抽象概念，用**重力场**类比**电场**：

| 重力场 | 电场 |
|--------|------|
| 有质量物体受拉向中心的力 | 带电粒子受吸引或排斥力 |
| 重力势能：与位置、质量有关 | 电势能：与位置、带电量有关 |
| **重力势**：与质量无关，是场的特性 | **电势**：与试探电荷无关，是场的特性 |

沿电场线方向，电场强度逐渐减弱，无穷远处为 0。

## 4. 电压

**概念**：又称**电势差**，是两点间电势的差值。

| 项目 | 说明 |
|------|------|
| **单位** | 伏特 (V) |
| **1V 含义** | 电场对 1C 电荷做 1J 的功 |
| **换算** | 1kV = 1000V |

---

## 5. 电阻

**概念**：材料或元器件对电流流动的**阻碍程度**。

| 项目 | 说明 |
|------|------|
| **单位** | 欧姆 (Ω) |
| **1Ω 含义** | 施加 1V 电压、电流为 1A 时，电阻为 1Ω |
| **换算** | 1KΩ = 1000Ω |

**电阻决定式**（了解）：

```
R = ρ · L / A
```

- R：电阻
- ρ：电阻率
- L：导体长度
- A：横截面积

> 后续会讲「电阻器」——专门用于控制电流的元器件，为电路提供电阻以限制电流。

## 6. 电路

| 术语 | 含义 |
|------|------|
| **电源** | 提供电能的装置 |
| **用电器** | 消耗电能的装置（灯泡、电动机等） |
| **电路** | 电源 + 用电器 + 导线 + 开关，构成电流可流过的路径 |

**电路状态**：

| 状态 | 说明 |
|------|------|
| **通路** | 正常接通，用电器正常工作 |
| **断路** | 某处被切断，无电流 |
| **短路** | 导线直接连电源正负极（**危险！**） |

## 7. 欧姆定律

**内容**：导体中电流与两端电压成正比，与电阻成反比。

```
I = U / R
```

| 符号 | 含义 | 单位 |
|------|------|------|
| U | 电压 | V |
| I | 电流 | A |
| R | 电阻 | Ω |

变形：`R = U/I`，`U = IR`

**决定式 vs 定义式**：

| | 决定式 | 定义式 |
|---|--------|--------|
| **用途** | 描述物理量由什么决定 | 描述「是什么」、如何测量 |
| **依赖** | 固有属性（长度、截面积、ρ） | 外部测量条件（U、I） |
| **电阻例** | R = ρ·L/A | R = U/I |

## 8. 直流电 VS 交流电

### 8.1 直流电 (DC)

- 有固定正负极，电流方向**始终不变**
- 举例：干电池、锂电池、光伏板

### 8.2 交流电 (AC)

- 电流方向随时间**周期性变化**
- 方向一定周期性变化，但大小不一定周期性变化
- 举例：家庭用电（220V-50Hz 正弦交流电）

<Diagram src="/electronic-components/diagrams/ac-dc.svg" alt="直流与交流波形对比" width={720} height={260} />

### 8.3 扩展内容（了解）

| 概念 | 说明 |
|------|------|
| **周期** | 完成一个完整波动循环的时间 |
| **零线与火线** | 零线电势始终为 0，火线电势不断变化 |
| **220V** | 交流电的**有效电压**——与相同电阻上产生相等热量的直流电压 |
| **峰值电压** | 一个周期内最大电压，220V 交流峰值 ≈ 311V |
| **50Hz** | 1 秒内 50 个周期，方向变化 100 次 |
| **整流** | AC → DC |
| **逆变** | DC → AC |

## 9. 弱电与强电

| | 弱电 | 强电 |
|---|------|------|
| **电压** | 一般较低，安全电压 ≤36V | 220V 家用电、1000V+ 高压 |
| **用途** | 3.3V/5V/12V 直流、信号传递 | 传递能量 |

> **安全提示**：弱电可直接触摸通常无生命危险；强电**绝对不能**用手触摸！

## 10. 家庭电路（了解）

**保护接地**目的：

1. 将零线电压锁定为地电位 (0V)
2. 故障时引导电流回地，触发断路器保护

**为什么摸火线危险？**

- 火线 220V，地 0V，身体与地构成回路 → 电流流过身体
- 路径：火线 → 你 → 地 → 零线 → 发电厂

> 家庭用电属于强电！禁止自行操作，禁止用身体验证结论。

## 11. 串联与并联

| | 串联 | 并联 |
|---|------|------|
| **连接** | 元件首尾相连，一条路径 | 元件并列，多条路径 |
| **电流** | 各处相等 | 各支路之和 = 总电流 |
| **电压** | 各元件电压之和 = 总电压 | 各支路电压相等 |

## 12. 电功率

**概念**：衡量电流在单位时间内所做的功，表示用电器消耗电能的快慢。

```
P = U · I
```

| 符号 | 单位 |
|------|------|
| P | 瓦特 (W)，1kW = 1000W |
| U | V |
| I | A |

**瓦时 vs 千瓦时**：

| 单位 | 含义 |
|------|------|
| **Wh（瓦时）** | 功率 × 时间，1W 用电器工作 1h = 1Wh |
| **kWh（千瓦时）** | 瓦时的千倍，俗称「度电」 |

**安时 / 毫安时**（电池容量）：

| 单位 | 含义 |
|------|------|
| **Ah** | 以 1A 电流持续工作 1h |
| **mAh** | Ah 的千分之一，手机/充电宝常用 |

## 13. 焦耳定律

**内容**：电流通过导体产生的热量与电流平方成正比，与电阻、通电时间成正比。

```
Q = I² · R · t
```

| 符号 | 单位 |
|------|------|
| Q | 焦耳 (J) |
| I | A |
| R | Ω |
| t | s |

**例题**：60Ω 电阻丝接 36V 电源，5s 内产生多少热量？

```
Q = (36V)² / 60Ω × 5s = 108J
```

> 只有**纯电阻电路**才能把电能完全用于产热。纯电阻电路可推导：  
> `P = U²/R = I²R`，`Q = UIt = Pt`
