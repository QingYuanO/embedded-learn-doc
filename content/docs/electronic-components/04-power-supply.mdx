## 3. 直流稳压电源

**直流稳压电源**将输入电源转换为**稳定直流输出**的设备，具备：

- 可调电压 / 电流
- 过载保护
- 恒压 (CV) / 恒流 (CC) 模式

是电子维修调试与**嵌入式系统供电**的核心设备。

### 典型用途

| 场景 | 说明 |
|------|------|
| 面包板实验 | 给 STM32 开发板、传感器模块供电 |
| 电路调试 | 逐步升压观察电路行为 |
| 充电测试 | 恒流模式模拟电池充电 |

### 使用注意

1. **先限压限流再通电**，避免烧毁被测电路
2. 确认极性：红线正极、黑线负极/地
3. 嵌入式常用电压：3.3V、5V、12V
4. 勿将输出直接短路（除非测试限流功能）
