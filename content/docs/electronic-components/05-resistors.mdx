> 电阻器是电子设备中最常见的元器件之一，主要作用是**阻碍电流流动**。

---

## 4.1 概述

**分类**：固定电阻、可变电阻、特种电阻。

> **特别注意**：电阻的「外观大小」和「阻值大小」**没有必然联系**！但外观大小往往与**功率**正相关——外观大的通常能承受更大功率。

**关键参数**：

| 参数 | 说明 |
|------|------|
| **阻值** | 越大阻碍电流能力越强 |
| **精度** | 一般 1% 或 5%，精密可达 0.01%，精度越高越贵 |
| **功率** | 长时间连续安全工作的最大功率 |

## 4.2 固定电阻

### 1. 线绕电阻

**工艺**：电阻丝（镍铬、锰铜等）绕在绝缘骨架（陶瓷）上，外层涂绝缘材料密封。

| 特点 | 说明 |
|------|------|
| 精度 | 最高 ±0.01% |
| 功率 | 可达数 kW |
| TCR | 约 20~200 ppm/℃ |
| 缺点 | 存在寄生电容/电感，**不适合高频**（WiFi、蓝牙、雷达等） |

**适用**：过载保护、大功率设备（大型机床电路）。

### 2. 碳膜电阻

**工艺**：高温分解结晶碳，蒸馏沉积在陶瓷棒上，刻螺旋槽控制阻值，外层涂环氧树脂。

| 特点 | 说明 |
|------|------|
| 精度 | 常用 ±5%，最高 ±1% |
| 功率 | 1/8W ~ 5W |
| TCR | 约 200~500 ppm/℃ |
| 温度特性 | **负温度系数**——温度升高阻值减小 |

**适用**：精度要求不高、控制成本的设备（电动玩具、老式电器）。

**识别**：通常为**土黄色**，**四色环**标记。

### 3. 金属膜电阻

**工艺**：高温真空镀金属膜（镍铬合金），刻蚀螺旋槽控制阻值。

| 特点 | 说明 |
|------|------|
| 精度 | 常用 ±1%，最高 ±0.01% |
| 功率 | 50mW ~ 20W |
| TCR | 约 50~100 ppm/℃ |

**适用**：精度要求略高的精密设备（万用表主板等）。

**识别**：通常为**天蓝色**，**五色环**标记。

### 4. 贴片电阻（薄膜、厚膜）

| 类型 | 厚度 | 精度 | 功率 | TCR | 适用 |
|------|------|------|------|-----|------|
| **厚膜** | ~100μm | 1%~5%，最高 0.1% | 10mW~10W | 50~200 ppm/℃ | 消费电子、汽车、家电（覆盖 90% 消费类） |
| **薄膜** | ~0.1μm | 0.1%~2%，最高 0.01% | 10mW~3W | 5~50 ppm/℃ | 精密仪器、高端音响、测量设备 |

> 几乎不能从外观区分薄膜与厚膜。

### 5. 金属箔电阻

**工艺**：合金与添加剂混合，涂几微米厚膜，刻蚀控制阻值。

| 特点 | 说明 |
|------|------|
| 精度 | 最高 ±0.001% |
| TCR | 极低，1 ppm/℃，特殊制造可达 0.05 ppm/℃ |
| 特性 | 低噪声、耐静电、低感低容 |

**所有电阻中精度和稳定性最好**，适用于航空航天、军工、医学等极高可靠性场景。

## 4.3 可调电阻

阻值可手动调节，多了调节指针。常见封装：微调式、多转式、旋转式、滑动式。

| 术语 | 说明 |
|------|------|
| **可调电阻** | 两个引脚即可，简单调节 |
| **电位器** | 三个引脚，调节电压比例 |

**适用**：音频调音量、照明调亮度。

## 4.4 特种电阻

### 1. 保险丝

电流高于额定值时**熔断**，保护后续电路。

| 类型 | 说明 |
|------|------|
| 一次性 | 熔断后需更换 |
| 自恢复 | 熔断后可复位反复使用 |

**关键参数**：额定电流、熔化特性（快速/延迟）、分断能力、耐压值。

### 2. 光敏电阻

阻值随光照强度变化——**黑暗中阻值高，光照下阻值小**。

| 参数 | 说明 |
|------|------|
| 亮电阻 | 10LUX 下测得 |
| 暗电阻 | 10LUX → 0LUX 后第 10 秒测得 |
| γ 值 | 阻值与光线变化系数，越大越敏感 |

**适用**：夜灯、路灯、光敏报警器（需配合三极管使用——夜晚亮白天灭的逻辑与光敏电阻特性相反）。

### 3. 压敏电阻 (MOV)

两端电压超过阈值时阻值**急剧下降**，实现过压保护。

| 特点 | 说明 |
|------|------|
| 材料 | 早期碳化硅，现多用氧化锌 |
| 限制 | 不能长时间冲击，寿命有限 |
| 适用 | 电源过压保护、防雷设备 |

**关键参数**：压敏电压、最大连续工作电压、最大限制电压。

## 4.5 阻值识别

### 1. 直标法

阻值、精度、额定功率直接标注在表面。

### 2. 三位数字代码

用于 ±5% 精度：前两位有效数字，第三位为加零个数。R 表示小数点。

| 标注 | 阻值 |
|------|------|
| 151 | 150Ω |
| 101 | 100Ω |

### 3. 四位数字代码

用于 ±1% 精度：前三位有效数字，第四位为加零个数。

| 标注 | 阻值 |
|------|------|
| 1001 | 1000Ω |
| 10R0 | 10.0Ω |

### 4. 精密标注法

用于 ±0.1% 精度，两位数字 + 一位字母。

| 标注 | 阻值 |
|------|------|
| 18D | 150KΩ |
| 01C | 10KΩ |

### 5. 色环法

4 或 5 个色环表示阻值和精度。因印刷工艺和色差，**实际中更常用万用表直接测量**。

## 4.6 阻值标准

为什么阻值看起来「奇葩」（2.7Ω、3.9Ω、6.8Ω）？因为要配合精度实现**阻值全覆盖**。

**E12 系列**（10% 精度）：阻值按指数间距分布，相邻设计值倍数约 **1.21**（10^(1/12) ≈ 1.21）。

还有 E6、E24、E48、E96 等系列，分别适用于不同精度。

## 4.7 元器件数据手册

数据手册 (Datasheet) 是选型和应用时的**核心参考资料**，由制造商提供，记录：

- 技术规格
- 功能描述
- 应用方法
- 可靠性数据

选型时务必查阅对应型号的数据手册。
