```
电子元器件入门
├── 第一章：电学基础        ← 电流、电压、电路、欧姆定律
├── 第二章：常见电子元器件  ← 封装、电阻、电容、电感、蜂鸣器、继电器
└── 第三章：半导体          ← PN 结、二极管、三极管、MOS 管
```

## 学习建议

1. **先打电学基础**：理解电流、电压、电阻的关系，再碰具体元器件。
2. **读第二章导读**：从 [常见电子元器件](/docs/electronic-components/02-common-components) 进入，按顺序学习各小节。
3. **无源器件优先**：电阻 → 电容 → 电感，三者是滤波、耦合、储能的基本积木。
4. **半导体放最后**：二极管、三极管、MOS 管建立在前述概念之上，与嵌入式 GPIO、驱动电路直接相关。
5. **动手验证**：配合万用表、面包板和 [立创商城](https://www.szlcsc.com/) 采购样品，比纯阅读更有效。
