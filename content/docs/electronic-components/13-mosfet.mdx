> MOS 管 (MOSFET) 通过**电场**控制电流，三极管通过**电流**控制电流。MOS 管是芯片的底层原理、芯片中的最小单位。

---

## 6.1 概述

**全称**：Metal Oxide Semiconductor Field Effect Transistor

| 对比 | 三极管 (BJT) | MOS 管 (FET) |
|------|-------------|-------------|
| 控制方式 | 电流控制电流 | **电压（电场）控制电流** |
| 输入阻抗 | 较低（kΩ 级） | **极高**（G 与沟道间有氧化绝缘层，pA 级漏电流） |
| 开关速度 | 较慢 | **更快** |
| 功耗 | 基极有持续电流 | 栅极几乎无电流（稳态） |
| 驱动 | 需持续基极电流 | 只需对栅电容充放电 |

应用：放大、开关、电压控制。嵌入式中 MOS 管常用于**电机驱动、电源切换、GPIO 驱动大电流**。

## 6.2 分类体系

### 6.2.1 总览

```
场效应管 (FET)
├── 结型 (JFET)
│   ├── N 沟道
│   └── P 沟道
└── 绝缘栅型 (MOSFET / IGFET)   ← 本文重点，嵌入式最常用
    ├── N 沟道
    │   ├── 增强型 (Enhancement)  ← 出场频率最高
    │   └── 耗尽型 (Depletion)
    └── P 沟道
        ├── 增强型
        └── 耗尽型
```

按两个维度划分：**沟道类型（N / P）** × **工作模式（增强 / 耗尽）**，共 4 种 MOSFET 组合。

### 6.2.2 JFET vs MOSFET

| 对比 | JFET（结型） | MOSFET（绝缘栅型） |
|------|-------------|-------------------|
| 栅极结构 | 栅极与沟道为 **PN 结** | 栅极与沟道间有 **SiO₂ 绝缘层** |
| 栅极阻抗 | 高，但**不能**正向偏置 PN 结 | **极高**，栅极可正、负偏置（在额定范围内） |
| 控制方式 | 反偏 PN 结，改变耗尽区宽度 | 电场感应形成 / 消除沟道 |
| 典型应用 | 模拟放大、高输入阻抗前端 | **数字开关、功率驱动、芯片内部** |

> 嵌入式 GPIO 驱动、电机 H 桥、电源管理几乎清一色用 **MOSFET**，JFET 多见于特殊模拟电路。

### 6.2.3 N 沟道 vs P 沟道

| | N 沟道 (NMOS) | P 沟道 (PMOS) |
|---|--------------|--------------|
| **沟道载流子** | 电子 | 空穴 |
| **衬底类型** | P 型衬底，嵌入 N 型 S / D | N 型衬底，嵌入 P 型 S / D |
| **增强型零偏** | V_GS = 0 时**截止** | V_GS = 0 时**截止** |
| **增强型导通** | V_GS > V_th（栅比源**更正**） | V_GS < −V_th（栅比源**更负**） |
| **典型开关位置** | **低端开关**（S 接 GND） | **高端开关**（S 接 VCC） |
| **GPIO 驱动** | 直接驱动（需 V_GS > V_th） | 需电平转换或 PMOS 专用驱动 |

### 6.2.4 增强型 vs 耗尽型

| | 增强型 (Enhancement) | 耗尽型 (Depletion) |
|---|---------------------|-------------------|
| **符号特征** | 沟道线为**三段虚线**（无沟道） | 沟道线为**一条实线**（有天然沟道） |
| **V_GS = 0 时** | **截止**（常关 OFF） | **导通**（常开 ON） |
| **N 沟道如何关断** | V_GS ≤ V_th 即截止 | 需加**负**栅压 V_GS < 0 才关断 |
| **N 沟道如何导通** | 需加**正**栅压 V_GS > V_th | V_GS = 0 已导通，正栅压更强 |
| **P 沟道如何关断** | V_GS ≥ −V_th 即截止 | 需加**正**栅压 V_GS > 0 才关断 |
| **P 沟道如何导通** | 需加**负**栅压 V_GS < −V_th | V_GS = 0 已导通 |
| **市场占比** | **绝对主流**（2N7002、IRF540 等） | 较少见（JFET 常具耗尽特性） |

**记忆口诀**：增强型 = 「**不加电不开**」；耗尽型 = 「**不加电也通**」。

### 6.2.5 四种 MOSFET 导通条件速查

| 类型 | 截止条件 | 导通条件 |
|------|---------|---------|
| N 沟道增强 | V_GS ≤ V_th | **V_GS > V_th** |
| N 沟道耗尽 | V_GS < V_off（负值） | V_GS ≥ V_off |
| P 沟道增强 | V_GS ≥ −V_th | **V_GS < −V_th** |
| P 沟道耗尽 | V_GS > V_off（正值） | V_GS ≤ V_off |

> 原书简写「N 沟道 V_GS > 0、P 沟道 V_GS < 0」指的是**栅极相对源极的正负关系**（V_G > V_S 或 V_G < V_S）。实际设计必须以数据手册中的 **V_th（阈值电压）** 为准——3.3V GPIO 驱动时，必须选 **V_th ≤ 2V** 的逻辑电平 MOS 管。

## 6.3 基本结构（N 沟道增强型）

基于 **P 型衬底**，嵌入两个 **N 型**区域作为**源极 (S)** 和**漏极 (D)**。

- **衬底 (Body / Substrate)**：P 型，出厂时与 **S 极内部相连**（外部不可见）
- 两个 N 区被 P 型衬底隔开，**V_GS = 0 时不形成沟道**
- 栅极 (G) 与衬底/沟道之间有一层 **SiO₂ 氧化绝缘层**，因此栅极输入阻抗极高

三个对外电极：**G（栅极）**、**S（源极）**、**D（漏极）**。

<Diagram src="/electronic-components/diagrams/nmos-structure.svg" alt="N沟道增强型MOS管结构" width={720} height={300} />

**P 沟道增强型**结构对称：N 型衬底 + 两个 P 型 S / D 区，理解 N 沟道后类推即可。

## 6.4 导通原理（以 N 沟道增强型为主）

### 6.4.1 正确导通条件

当 **V_GS > V_th**（栅极电压比源极高出阈值以上）时，P 衬底表面感应出 N 型**反型层**（沟道），N 沟道 MOSFET **导通**，电流从 **D → S**（电子实际从 S 流向 D）。

```
V_GS > V_th  →  沟道形成  →  D→S 导通（常规电流方向）
V_GS ≤ V_th  →  沟道消失  →  截止
```

### 6.4.2 两种常见错误接法

原书图 1、图 2 说明了两类典型误用（此处用文字还原）：

**错误 1 — 栅源电压不足，无法导通**

```
        VCC
         │
        [负载]
         │
    D ───┤
         │  N-MOS
    S ───┤
         │
        GND ─── G（栅极也接 GND）
```

栅极与源极同电位 → **V_GS = 0 < V_th** → 沟道无法形成 → **不导通**。

**错误 2 — 体二极管主导，无法控制开关**

```
        VCC
         │
    S ───┤
         │  N-MOS
    D ───┤
         │
        [负载]
         │
        GND    G（即使关断）
```

N 沟道管做**高端开关**：S 接 VCC，D 电位低于 S → 内置 **S→D 体二极管正向偏置** → 电流经二极管流通 → **栅极失去控制**。

### 6.4.3 正确接法 — 低端开关

```
        VCC
         │
        [负载]
         │
    D ───┤
         │  N-MOS
    S ───┤
         │
        GND

    G ── GPIO（高电平时 V_GS ≈ V_CC > V_th → 导通）
```

- S 接 **GND（电位最低）** → 导通时 D > S，**体二极管反偏**
- GPIO 输出高电平 → V_GS 足够 → 沟道导通

### 6.4.4 体二极管（寄生二极管）

因衬底与 S 极相连，N 沟道 MOSFET 内部存在一只 **S → D** 方向的体二极管（阳极在 S，阴极在 D），与沟道导通方向相反：

| 状态 | D 与 S 电位 | 体二极管 |
|------|------------|---------|
| 低端开关导通 | D > S | **反偏** ✓ |
| 高端误用 N-MOS | D < S | **正偏** ✗，不受栅极控制 |
| 电机续流 | 电感反向电动势 | 需正确方向吸收，或并联外部二极管 |

> **技巧**：使用 MOS 管时，**让体二极管处于反偏**——这是选型与接线时首先要检查的。

## 6.5 快速分辨 MOS 管

### 6.5.1 看电路符号（原理图）

| 规则 | 说明 |
|------|------|
| **1. 栅极 G** | 符号**中间**竖线连接的那一端 |
| **2. 源极 S** | 符号上**线最多**的一端（含沟道线、衬底线、体二极管线） |
| **3. 漏极 D** | 剩下的一端 |
| **4. 沟道类型** | 源极箭头**指向 G**（往里指）→ **N 沟道**；**背离 G**（往外指）→ **P 沟道** |
| **5. 增强 / 耗尽** | G 对面：**三段虚线** = 增强型；**一条实线** = 耗尽型 |
| **6. 导通条件** | N 增强：V_GS > V_th；P 增强：V_GS < −V_th |
| **7. 电流方向** | 接线时让**体二极管反偏** |

### 6.5.2 看封装引脚（实物）

符号规则**不适用于**物理引脚排列——不同封装、不同厂家的 G / D / S 顺序可能不同：

| 封装 | 常见引脚顺序（面向文字正面，从左到右） |
|------|----------------------------------------|
| TO-220 / TO-247 | 常见 G-D-S 或 G-S-D，**以数据手册为准** |
| SOT-23 | 多种排列，需查手册或万用表二极管档测体二极管定位 S / D |
| SO-8 | 散热焊盘常接 D 极 |

> 实物识别：**用万用表二极管档**，表笔接 G-S 应无导通；接 S-D 会测到体二极管压降（约 0.5~0.7V），据此可区分 S 与 D。

## 6.6 嵌入式典型应用

### 低端开关（N 沟道增强）

```
GPIO ──[Rg]── G
              S ── GND
              D ── 负载 ── VCC
```

- 3.3V / 5V GPIO 可直接驱动 **逻辑电平** NMOS
- Rg（100Ω~10kΩ）限制栅极充放电尖峰电流

### 高端开关（P 沟道增强）

```
GPIO ──[Rg]── G
              S ── VCC
              D ── 负载 ── GND
```

- GPIO **低电平** → V_GS < 0 → PMOS 导通
- 适合电源路径切换（如 USB / 电池切换）

### 半桥 / H 桥

N-MOS 做高端需 **自举电路** 或 **专用栅极驱动 IC**（如 IR2104），因为导通时 S 极电压升高，V_GS 可能不足。

---

## 6.7 嵌入式常用型号

| 型号 | 类型 | V_th / 驱动 | 典型用途 |
|------|------|------------|---------|
| 2N7002 / SI2302 | N 增强 | 逻辑电平，~1.5V | 3.3V / 5V GPIO 驱动小负载（200mA 以下） |
| IRLZ44N | N 增强 | 逻辑电平，~2V | 大电流低端开关（GPIO 可驱动） |
| IRF540N | N 增强 | 标准型，需 ~10V V_GS | 电机、大电流，**需 10~12V 栅极驱动** |
| AO3401 | P 增强 | 逻辑电平 | 高端电源切换 |

> **注意**：IRF540 与 IRLZ44N 虽然都是 N 沟道增强型，但 IRF540 是**标准栅** MOSFET，3.3V GPIO **无法**将其完全导通；IRLZ44N 才是**逻辑电平**型，可直接由 MCU 驱动。

## 6.8 小结

| 要点 | 内容 |
|------|------|
| 最常用 | **N 沟道增强型 MOSFET** |
| 控制量 | **V_GS**（栅源电压），不是栅极绝对电压 |
| 导通判据 | N 增强：**V_GS > V_th**；P 增强：**V_GS < −V_th** |
| 接线原则 | N-MOS 低端开关，P-MOS 高端开关；**体二极管反偏** |
| 与 BJT 区别 | 电压控、输入阻抗高、开关快、栅极几乎不耗直流功率 |

> N 沟道增强型理解透彻后，JFET、P 沟道、耗尽型只需对照符号与导通条件表即可触类旁通。
