## 5.1 何为电容？

**电容** = 「储电荷的容器」。两个彼此绝缘的极板，中间夹有电介质。施加电压时极板积累电荷，核心功能是**充电、放电**。

| 对比 | 电容 | 电池 |
|------|------|------|
| 放电 | **瞬间**放电 | 化学能缓慢释放 |
| 充电 | 速度逐渐变慢，电压与外部一致时停止 | — |

**电容决定式**（了解）：

```
C = ε · A / D
```

- ε：介质介电常数 (F/m)
- A：极板有效面积 (m²)
- D：极板间距 (m)

| 项目 | 说明 |
|------|------|
| **单位** | 法拉 (F) |
| **1F 含义** | 加 1V 电压能储存 1C 电荷 |
| **换算** | 1F = 1000mF，1mF = 1000μF |

## 5.3 电容的特点

### 1. 充电与放电

- **刚充电**：电压≈0，对电流阻碍极小，电流很大
- **快充满**：电压接近电源，电难压入，阻碍极大，电流很小
- **完全充满**：电压 = 电源电压，相当于**开路**

### 2. 隔直流、通交流

- 直流无法通过（充满后开路）
- 交流可以「通过」（实际是反复充放电）

<Diagram src="/electronic-components/diagrams/cap-ac-dc.svg" alt="电容隔直流通交流" width={720} height={220} />

### 3. 阻抗的概念

| 类型 | 说明 |
|------|------|
| **电阻** | 固定阻碍，能量耗散（发热） |
| **容抗** | 电容的动态阻碍，**频率越高越小** |
| **感抗** | 电感的动态阻碍，**频率越高越大** |

**容抗公式**（了解）：

```
Xc = 1 / (2πfC)
```

规律：

1. 频率不变 → 容值越小，容抗越大
2. 容值不变 → 频率越小，容抗越大

> 对比电阻与电容：电阻会发热，电容不发热。交流风扇常用电容限流调速。

### 4. 滤波作用

**整流之后**，电压往往仍是「一高一低」的脉动直流，还不能直接给精密电路用。此时在负载两端**并联一个大电容**（通常是电解电容），利用充放电把波形「垫平」——这就是**电容滤波**。

```
脉动直流（整流后）     电容滤波后
    ∧   ∧   ∧              ─────────
   / \ / \ / \      →      相对平稳的直流
  /   V   V   \
```

**工作过程**：

1. **电压上升（接近峰值）**：电容充电，把多余电荷存起来
2. **电压下降（介于两个峰值之间）**：二极管截止，电容向负载**放电**，补上电压缺口
3. 下一个峰值到来，再次充电……循环往复

> 类比：水库在雨季蓄水、旱季放水，让下游水量更稳定。电容就是电路里的「小水库」。

电容越大，放电越慢，输出电压越平稳。所以电源滤波常用 **100μF ~ 1000μF** 的电解电容；MCU 板子上每个芯片旁的 **0.1μF** 则是滤除高频噪声（去耦），原理类似但目的不同。

### 5. 全波整流与电容滤波

#### 5.1 先分清两件事

| 环节 | 谁来做 | 作用 |
|------|--------|------|
| **整流** | 二极管（通常 4 个组成桥式） | 把交流变成**单向**的脉动直流 |
| **滤波** | 电容（并联在负载上） | 把脉动直流**平滑**成较稳定的直流 |

原文把二者放在电容章节，是因为：**没有电容配合，全波整流输出的仍不是「干净」的直流**——电容是全波整流电路里不可或缺的一环。

#### 5.2 什么是整流？

**整流** = 把交流电 (AC) 转换成直流电 (DC)。

把电流想象成双向车道上的车流：

- **交流**：车一会儿从左往右，一会儿从右往左（方向周期性变化）
- **直流**：车只朝一个方向开

二极管只允许单方向导通，因此可用来「强制」电流只走一个方向。

<Diagram src="/electronic-components/diagrams/rectifier.svg" alt="半波与全波整流波形对比" width={720} height={240} />

#### 5.3 半波整流 vs 全波整流

| | 半波整流 | 全波整流 |
|---|---------|---------|
| **二极管数量** | 1 个 | 4 个（桥式）或 2 个 + 中心抽头变压器 |
| **利用的波形** | 仅正半周 | **正、负半周都翻转**成同一方向 |
| **输出频率** | 与输入相同（50Hz → 50Hz 脉动） | 脉动频率翻倍（50Hz → 100Hz 脉动） |
| **效率** | 低，一半能量被丢弃 | **高**，正负半周都贡献能量 |
| **电容滤波难度** | 间隔长，需更大电容 | 脉动更密，相对更易滤平 |

**全波整流**像一座立交桥：无论车从哪个方向进来，出口都统一成一个方向，因此比半波整流**效率高得多**。家用适配器、线性电源几乎都用全波整流。

#### 5.4 典型全波整流 + 电容滤波电路

```
        交流输入（如 220V→变压器→12V AC）
              │
              ▼
    ┌─────────────────────┐
    │   桥式整流（4 二极管）  │  ← 整流：AC → 脉动 DC
    └──────────┬──────────┘
               │
         ┌─────┴─────┐
         │  电解电容 C  │  ← 滤波：脉动 DC → 较平稳 DC
         └─────┬─────┘
               │
               ▼
            负载 R
          （如 5V 模块）
```

**波形变化**（从时间轴上看）：

```
① 交流输入        ～～∿∿～～∿∿～～        正负交替
② 全波整流后      ∧∧∧∧∧∧∧∧∧∧        全为正，但有起伏（脉动直流）
③ 电容滤波后      ──∿∿──∿∿──∿∿──        驼峰被「垫平」，仍有小纹波
```

1. **桥式整流输出**：一串向上的「驼峰」，峰顶 ≈ 输入峰值 − 二极管压降（4 个二极管约 1.4V）
2. **电容充电**：每个驼峰到来时，电容电压被拉到接近峰值
3. **电容放电**：驼峰之间的低谷，由电容向负载供电，电压缓慢下降
4. **纹波 (Ripple)**：输出仍会有小幅起伏；电容越大、负载越小，纹波越小

#### 5.5 电容在全波整流中的关键参数

| 参数 | 为什么重要 |
|------|-----------|
| **容值 C** | 越大储能越多，放电越慢，纹波越小 |
| **耐压** | 必须 **> 输出峰值电压**（如 12V AC 整流后峰值约 17V，电容至少选 25V） |
| **类型** | 滤波用大容量**电解电容**；极性不能接反，否则会爆炸 |
| **ESR** | 等效串联电阻越小，高频纹波抑制越好 |

**粗算纹波**（了解）：

```
ΔV ≈ I_load / (f × C)
```

- `I_load`：负载电流
- `f`：脉动频率（全波整流为 2 × 工频，50Hz 时 f = 100Hz）
- `C`：滤波电容

例：负载 100mA，C = 470μF，全波 100Hz → ΔV ≈ 0.1 / (100 × 470×10⁻⁶) ≈ **2.1V** 纹波

#### 5.6 和嵌入式开发的关系

- 开发板上的 **DC 插座 / USB** 输入，内部往往就是「变压器 → 桥式整流 → 大电解电容 → LDO/DC-DC」
- 板载 **100μF + 0.1μF** 组合：大电容滤低频纹波，小陶瓷电容滤高频
- 若示波器测电源脚看到周期性锯齿，多半是**滤波电容不足或失效**

> 二极管的整流原理详见 [二极管 · 整流](/docs/electronic-components/11-diodes#42-整流二极管)。本章重点：**整流负责改方向，电容负责填低谷**。

## 5.4 常见电容器

### 1. 陶瓷电容

- 以陶瓷为介质，通常**无极性**
- 电子设备中使用最多
- **适用**：高频电路、噪声去除、滤波

### 2. 电解电容

- 内部填充电解质，**有极性**
- 容值大（0.5pF ~ 10000μF），额定电压 2.5V ~ 500V
- 储能密度高，但电解质会蒸发影响寿命
- **不能直接在交流电中使用**
- **适用**：低频直流滤波、闪光灯、电机启动

### 3. 空气电容

- 以空气为介质，通常为**可调电容**
- 容值小（100pF ~ 1nF），抗磁性好
- **适用**：高频调谐（无线电、广播）、医疗设备

### 4. 超级电容

- 容量极大，充放电极快
- 分类：静电双层、赝电容、混合电容
- **适用**：UPS、新能源汽车动能回收

| 对比 | 超级电容 | 电池 |
|------|---------|------|
| 能量密度 | 较低 | 高 |
| 功率密度 | 极高 | 较低 |
| 寿命 | 充放电次数多 | 有限 |

## 5.5 容值识别

### 1. 直接标注法

电解电容常用，容值直接标在表面。

### 2. 乘方数表示法

陶瓷电容三位数字：首两位有效数字，第三位为加零个数，单位 **pF**。

### 3. 字母表示法

字母 p、n、u、m 分别表示 pF、nF、μF、mF。字母前为整数，字母后为小数。

### 4. 贴片电解电容

指标法标注，容值直接标注。
