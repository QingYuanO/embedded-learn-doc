## 本章导读

第二章按「先认识环境，再逐个元器件」的顺序展开：

```
常见电子元器件
├── 封装与采购      ← 怎么买、怎么焊
├── 直流稳压电源    ← 实验台供电
├── 电阻器          ← 最基础的无源器件
├── 电容器          ← 储能、滤波、耦合
├── 电感器          ← 储能、滤波（与电容互补）
└── 蜂鸣器与继电器  ← 常见的机电输出
```

## 无源 vs 机电

| 类别 | 本章涉及 | 核心作用 |
|------|----------|----------|
| **无源器件** | 电阻、电容、电感 | 不主动放大或产生能量，通过物理特性影响电路 |
| **机电器件** | 蜂鸣器、继电器 | 电信号 ↔ 声音 / 机械开关 |
| **实验设备** | 直流稳压电源 | 为电路提供可调、稳定的直流电 |

> 二极管、三极管、MOS 管等**半导体器件**在 [第三章](/docs/electronic-components/09-semiconductor-basics) 讲解。

## 学习顺序建议

1. **封装与采购** — 知道元件长什么样、从哪里买，后续实验才顺。
2. **电阻 → 电容 → 电感** — 三大无源器件，嵌入式电路里无处不在（上拉、滤波、储能）。
3. **蜂鸣器与继电器** — 与 MCU GPIO 直接相关，是常见的输出方式。
4. 每学一种器件，对照 [数据手册](/docs/electronic-components/05-resistors#47-元器件数据手册) 看关键参数，比死记型号更有效。

## 与嵌入式的关系

| 元器件 | 嵌入式典型用途 |
|--------|---------------|
| 电阻 | GPIO 上拉/下拉、LED 限流、分压采样 |
| 电容 | 电源去耦（0.1μF + 10μF）、复位 RC |
| 电感 | DC-DC 开关电源、EMI 滤波 |
| 蜂鸣器 | 按键提示音（GPIO + PWM） |
| 继电器 | 小 MCU 控制大电流负载（电机、灯） |
