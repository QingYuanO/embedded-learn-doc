## 简介
Universal Asynchronous Receiver Transmitter 即通用异步收发器，是一种通用的**串行**、**异步**通信总线。该总线有两条数据线，可以实现全双工的发送和接收。
在嵌入式系统中常用于主机与辅助设备之间的通信。

## UART 数据帧结构


1. 空闲位
- 线路处于高电平，表示没有数据传输。
2. 起始位
- 1位，低电平（0），表示数据传输的开始。
3. 数据位
- 5~8位（可配置），用于传输实际的数据。
- 先发送低位，后发送高位。
4. 校验位
- 可选，有或无。用于检测数据传输过程中是否出错。
- 奇偶校验：
  - 奇校验：数据位和校验位中"1"的个数为奇数
  - 偶校验：数据位和校验位中"1"的个数为偶数
  - 例如：数据位为 1010（两个1），若为奇校验则校验位为1（使总1的个数为3），若为偶校验则校验位为0（使总1的个数为2）
5. 停止位
- 1位、1.5位或2位高电平（1），表示一帧数据的结束。
