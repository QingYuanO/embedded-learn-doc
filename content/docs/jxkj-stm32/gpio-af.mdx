> 面向 **STM32F103** + 标准库。F103 **没有** F4 那种 AF0～AF15 寄存器选通，而是 **默认引脚固定 + AFIO 重映射** 切换备用引脚。  
> 以 **LQFP48（F103C8T6）** 为主；标注 **64pin+** 表示更大封装才有该脚。


## 一、怎么用这张表

1. **查默认**：先找外设默认引脚（如 USART1 → PA9/PA10）。  
2. **要改脚**：开 `RCC_APB2Periph_AFIO` → `GPIO_PinRemapConfig()` → GPIO 配 **复用模式** `AF_PP` / `AF_OD`。  
3. **同一引脚** 同一时刻只能给一个外设用（如 PA0 不能同时 TIM2_CH1 和 TIM5_CH1）。  
4. **TIM 重映射** 详见 [TIM 引脚重映射](../jxkj-stm32/tim/timer-guide)。

| 符号 | 含义 |
|------|------|
| **默认** | 不调用 Remap 时的映射 |
| **Remap** | 调用 `GPIO_PinRemapConfig(xxx, ENABLE)` 后的映射 |
| **—** | 该封装无此引脚，或该功能不可用 |


## 二、AFIO 重映射总表（标准库宏）

| 标准库宏 | 影响外设 | 默认引脚 | 重映射后引脚 |
|----------|----------|----------|--------------|
| `GPIO_Remap_SPI1` | SPI1 | PA4 NSS, PA5 SCK, PA6 MISO, PA7 MOSI | PB3 SCK, PB4 MISO, PB5 MOSI（NSS 仍常用软件 CS） |
| `GPIO_Remap_I2C1` | I2C1 | PB6 SCL, PB7 SDA | PB8 SCL, PB9 SDA |
| `GPIO_Remap_USART1` | USART1 | PA9 TX, PA10 RX | PB6 TX, PB7 RX |
| `GPIO_Remap_USART2` | USART2 | PA2 TX, PA3 RX | PD5 TX, PD6 RX（**64pin+**） |
| `GPIO_PartialRemap_USART3` | USART3 | PB10 TX, PB11 RX, PB12 CK, PB13 CTS, PB14 RTS | PC10 TX, PC11 RX, PC12 CK, PD11 CTS, PD12 RTS（部分脚 **64pin+**） |
| `GPIO_FullRemap_USART3` | USART3 | 同上默认 | PD8 TX, PD9 RX, PD10 CK, PD11 CTS, PD12 RTS（**64pin+**） |
| `GPIO_PartialRemap_TIM1` | TIM1 | 见 TIM 表 | CH1N→PA7, CH2N→PB0, CH3N→PB1, BKIN→PA6 |
| `GPIO_PartialRemap1_TIM2` | TIM2 | PA0～PA3 | CH1→PA15, CH2→PB3, CH3/4 不变 |
| `GPIO_PartialRemap2_TIM2` | TIM2 | PA0～PA3 | CH3→PB10, CH4→PB11 |
| `GPIO_FullRemap_TIM2` | TIM2 | PA0～PA3 | CH1→PA15, CH2→PB3, CH3→PB10, CH4→PB11 |
| `GPIO_PartialRemap_TIM3` | TIM3 | PA6/7, PB0/1 | CH1→PB4, CH2→PB5, CH3/4 仍为 PB0/1 |
| `GPIO_FullRemap_TIM3` | TIM3 | PA6/7, PB0/1 | CH1～4 → PC6～PC9（**64pin+**） |
| `GPIO_Remap_TIM4` | TIM4 | PB6～PB9 | PD12～PD15（**64pin+**） |
| `GPIO_Remap_CAN1` | CAN | PA11 RX, PA12 TX | PB8 RX, PB9 TX |
| `GPIO_Remap_PD01` | OSC | PD0/PD1 作 GPIO | 释放 PD0/PD1 为普通 IO（**64pin+**，失去外部晶振脚） |
| `GPIO_Remap_SWJ_NoJTRST` | 调试 | PB4 为 NJTRST | 释放 PB4 作 GPIO（保留 SWD） |
| `GPIO_Remap_SWJ_JTAGDisable` | 调试 | PA15/PB3/PB4 为 JTAG | 释放三脚作 GPIO（保留 SWD：PA13/14） |
| `GPIO_Remap_SWJ_Disable` | 调试 | 关闭 JTAG+SWD | 释放 PA13/14/15、PB3/4（**无法再调试**） |
| `GPIO_Remap_SPI3` | SPI3 | — | 与 SPI1 Remap 共用 PB3/4/5（**高密度**） |
| `GPIO_Remap_TIM5CH4_LSI` | TIM5 | — | CH4 与 LSI 内部相连（特殊用途） |

> **互斥**：同一组 Remap 宏不能乱叠；TIM2 三种 Remap 三选一。以参考手册 *Alternate function remapping* 为准。

## 四、按外设：默认引脚速查

### 4.1 USART

| 外设 | TX | RX | CK | CTS | RTS | 重映射 |
|------|----|----|----|----|-----|--------|
| **USART1** | PA9 | PA10 | PA8 | PA11 | PA12 | TX/RX → PB6/PB7 |
| **USART2** | PA2 | PA3 | PA4 | PA0 | PA1 | TX/RX → PD5/PD6 |
| **USART3** | PB10 | PB11 | PB12 | PB13 | PB14 | Partial → PC10～PC12 等；Full → PD8～PD12 |

### 4.2 SPI

| 外设 | NSS | SCK | MISO | MOSI | 重映射 |
|------|-----|-----|------|------|--------|
| **SPI1** | PA4 | PA5 | PA6 | PA7 | SCK/MISO/MOSI → PB3/PB4/PB5 |
| **SPI2** | PB12 | PB13 | PB14 | PB15 | 无 Remap |

### 4.3 I2C

| 外设 | SCL | SDA | 重映射 |
|------|-----|-----|--------|
| **I2C1** | PB6 | PB7 | → PB8/PB9 |
| **I2C2** | PB10 | PB11 | 无 Remap |

### 4.4 CAN

| 信号 | 默认 | Remap |
|------|------|-------|
| **CAN_RX** | PA11 | PB8 |
| **CAN_TX** | PA12 | PB9 |

### 4.5 ADC（模拟输入，非数字复用）

| 通道 | 默认引脚 |
|------|----------|
| ADC1_IN0～IN3 | PA0～PA3 |
| ADC1_IN4～IN7 | PA4～PA7 |
| ADC1_IN8～IN9 | PB0～PB1 |
| ADC1_IN10～IN15 | PC0～PC5（**64pin+**） |
| 内部 | 温度传感器、VREFINT |

配置为 **`GPIO_Mode_AIN`**，不走 AF_PP。

### 4.6 TIM（摘要）

| 定时器 | 默认 CH 引脚 | 重映射要点 |
|--------|--------------|------------|
| **TIM1** | PA8～11, 互补 PB13～15, BKIN PB12 | Partial：互补改 PA7/PB0/PB1，BKIN→PA6 |
| **TIM2** | PA0～PA3 | 三种 Remap，ETR 固定 PA0 |
| **TIM3** | PA6/7, PB0/1 | Partial→PB4/5；Full→PC6～9 |
| **TIM4** | PB6～PB9 | → PD12～15 |
| **TIM5** | PA0～PA3 | 无 Remap，与 TIM2 冲突 |

完整 TIM 引脚表见 [TIM 定时器详解 · 引脚对照](../jxkj-stm32/tim/timer-guide#142-tim1高级定时器)。


## 五、F103C8（LQFP48）实用引脚一览

48 脚封装 **实际可用 GPIO** 约 37 个，规划外设时注意：

| 引脚 | 常见用途 | 冲突提醒 |
|------|----------|----------|
| PA0～PA3 | TIM2 / TIM5 / USART2 / ADC | TIM2 与 TIM5 不能同时用 |
| PA4～PA7 | SPI1 / TIM3 / ADC | 最小系统 SPI Flash 常用 |
| PA9/10 | USART1 |  USB 串口模块默认 TX/RX |
| PA13/14 | SWD 调试 | 勿当普通 IO（除非关 SWD） |
| PB6/7 | I2C1 / TIM4 | MPU6050、OLED 常用 |
| PB10/11 | I2C2 / USART3 | |
| PB12～15 | SPI2 / TIM1 互补 | |
| PC13 | LED / 按键 | 接 32 kHz 晶振时慎用 PC14/15 |
| PD0/1 | 8 MHz 晶振 | 无外部晶振时可 Remap 作 GPIO |

**无 USB 引脚功能** 的型号（如 F103C8）：PA11/12 的 USBDM/USBDP **不可用**，可当 CAN 或 GPIO。


## 六、配置示例：USART1 重映射到 PB6/PB7

```c
RCC_APB2PeriphClockCmd(RCC_APB2Periph_GPIOB | RCC_APB2Periph_AFIO, ENABLE);
RCC_APB2PeriphClockCmd(RCC_APB2Periph_USART1, ENABLE);

GPIO_PinRemapConfig(GPIO_Remap_USART1, ENABLE);

GPIO_InitTypeDef init;
init.GPIO_Pin   = GPIO_Pin_6 | GPIO_Pin_7;
init.GPIO_Mode  = GPIO_Mode_AF_PP;
init.GPIO_Speed = GPIO_Speed_50MHz;
GPIO_Init(GPIOB, &init);

/* USART_Init(USART1, ...) */
```


## 七、常见冲突

| 冲突组 | 涉及引脚 | 建议 |
|--------|----------|------|
| TIM2 vs TIM5 | PA0～PA3 | 只启用一个 |
| TIM3 默认 vs TIM8 | PA6/7, PB0/1 | 低密度无 TIM8，可忽略 |
| SPI1 Remap vs JTAG | PB3/4/5 | 需 `GPIO_Remap_SWJ_JTAGDisable` 释放 PB3/4 |
| I2C1 默认 vs TIM4 | PB6/7 | 二选一或 I2C1 Remap 到 PB8/9 |
| USART1 Remap vs I2C1 | PB6/7 | 二选一 |
| TIM3 Full vs TIM8 | PC6～9 | 64pin+ 才存在 |

## 八、小结

| 要点 | 说明 |
|------|------|
| F103 复用 | **默认引脚 + AFIO Remap**，不是 AF 编号选择 |
| 必开 AFIO | Remap、EXTI 引脚选择前开 `RCC_APB2Periph_AFIO` |
| GPIO 模式 | 外设输出用 `AF_PP`/`AF_OD`；ADC 用 `AIN` |
| 查表顺序 | 外设默认 → 是否 Remap → 封装有无该脚 |
| 权威来源 | STM32F10xxx 参考手册 **Alternate function I/O** 章节 |
