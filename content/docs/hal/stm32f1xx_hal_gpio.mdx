# STM32F1xx HAL GPIO 驱动

## 概述

STM32F1xx HAL GPIO驱动提供了对STM32F1系列微控制器通用输入输出(GPIO)端口的完整控制功能。该驱动支持GPIO的初始化、配置、读写操作以及外部中断处理。

## 主要特性

- **多模式支持**: 输入、输出、复用功能、模拟模式
- **中断处理**: 支持上升沿、下降沿、双边沿触发的外部中断
- **事件处理**: 支持外部事件模式
- **上拉/下拉配置**: 可配置内部上拉或下拉电阻
- **输出类型**: 推挽输出和开漏输出
- **速度控制**: 低、中、高速配置

## 数据结构

### GPIO_InitTypeDef

GPIO初始化结构体，用于配置GPIO引脚参数：

```c
typedef struct
{
  uint32_t Pin;       /*!< 指定要配置的GPIO引脚 */
  uint32_t Mode;      /*!< 指定选中引脚的工作模式 */
  uint32_t Pull;      /*!< 指定选中引脚的上拉或下拉激活 */
  uint32_t Speed;     /*!< 指定选中引脚的速度 */
} GPIO_InitTypeDef;
```

### GPIO_PinState

GPIO引脚状态枚举：

```c
typedef enum
{
  GPIO_PIN_RESET = 0u,  /*!< 引脚低电平 */
  GPIO_PIN_SET          /*!< 引脚高电平 */
} GPIO_PinState;
```

#define GPIO_PIN_0                 ((uint16_t)0x0001)  /* 引脚0 */

## 工作模式

### GPIO模式定义

| 模式 | 值 | 描述 |
|------|----|----|
| `GPIO_MODE_INPUT` | 0x00000000 | 浮空输入模式 |
| `GPIO_MODE_OUTPUT_PP` | 0x00000001 | 推挽输出模式 |
| `GPIO_MODE_OUTPUT_OD` | 0x00000011 | 开漏输出模式 |
| `GPIO_MODE_AF_PP` | 0x00000002 | 复用功能推挽模式 |
| `GPIO_MODE_AF_OD` | 0x00000012 | 复用功能开漏模式 |
| `GPIO_MODE_ANALOG` | 0x00000003 | 模拟模式 |

### 中断模式

| 模式 | 值 | 描述 |
|------|----|----|
| `GPIO_MODE_IT_RISING` | 0x10110000 | 上升沿外部中断 |
| `GPIO_MODE_IT_FALLING` | 0x10210000 | 下降沿外部中断 |
| `GPIO_MODE_IT_RISING_FALLING` | 0x10310000 | 双边沿外部中断 |

### 事件模式

| 模式 | 值 | 描述 |
|------|----|----|
| `GPIO_MODE_EVT_RISING` | 0x10120000 | 上升沿外部事件 |
| `GPIO_MODE_EVT_FALLING` | 0x10220000 | 下降沿外部事件 |
| `GPIO_MODE_EVT_RISING_FALLING` | 0x10320000 | 双边沿外部事件 |

## 核心函数

### 初始化和反初始化

#### HAL_GPIO_Init
```c
void HAL_GPIO_Init(GPIO_TypeDef *GPIOx, GPIO_InitTypeDef *GPIO_Init);
```
**功能**: 根据GPIO_InitStruct中的指定参数初始化GPIOx外设
**参数**:
- `GPIOx`: GPIO端口，其中x可以是(A..G)
- `GPIO_Init`: 指向GPIO_InitTypeDef结构的指针

#### HAL_GPIO_DeInit
```c
void HAL_GPIO_DeInit(GPIO_TypeDef *GPIOx, uint32_t GPIO_Pin);
```
**功能**: 反初始化GPIOx外设寄存器到其默认复位值
**参数**:
- `GPIOx`: GPIO端口，其中x可以是(A..G)
- `GPIO_Pin`: 要反初始化的GPIO引脚

### IO操作函数

#### HAL_GPIO_ReadPin
```c
GPIO_PinState HAL_GPIO_ReadPin(GPIO_TypeDef *GPIOx, uint16_t GPIO_Pin);
```
**功能**: 读取指定的输入端口引脚
**返回值**: 输入端口引脚值(GPIO_PIN_RESET或GPIO_PIN_SET)

#### HAL_GPIO_WritePin
```c
void HAL_GPIO_WritePin(GPIO_TypeDef *GPIOx, uint16_t GPIO_Pin, GPIO_PinState PinState);
```
**功能**: 设置或清除选定的数据端口位
**参数**:
- `PinState`: 指定要写入选中引脚的值

#### HAL_GPIO_TogglePin
```c
void HAL_GPIO_TogglePin(GPIO_TypeDef *GPIOx, uint16_t GPIO_Pin);
```
**功能**: 切换指定的GPIO引脚

#### HAL_GPIO_LockPin
```c
HAL_StatusTypeDef HAL_GPIO_LockPin(GPIO_TypeDef *GPIOx, uint16_t GPIO_Pin);
```
**功能**: 锁定GPIO引脚配置寄存器
**返回值**: HAL状态

### 外部中断处理

#### HAL_GPIO_EXTI_IRQHandler
```c
void HAL_GPIO_EXTI_IRQHandler(uint16_t GPIO_Pin);
```
**功能**: 处理GPIO外部中断请求

#### HAL_GPIO_EXTI_Callback
```c
void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin);
```
**功能**: 外部中断回调函数，用户需要重写此函数

## 宏定义

### EXTI标志位操作

```c
// 检查指定的EXTI线标志是否设置
#define __HAL_GPIO_EXTI_GET_FLAG(__EXTI_LINE__) (EXTI->PR & (__EXTI_LINE__))

// 清除EXTI线的待处理标志
#define __HAL_GPIO_EXTI_CLEAR_FLAG(__EXTI_LINE__) (EXTI->PR = (__EXTI_LINE__))

// 检查指定的EXTI线是否断言
#define __HAL_GPIO_EXTI_GET_IT(__EXTI_LINE__) (EXTI->PR & (__EXTI_LINE__))

// 清除EXTI线的待处理位
#define __HAL_GPIO_EXTI_CLEAR_IT(__EXTI_LINE__) (EXTI->PR = (__EXTI_LINE__))

// 在选定的EXTI线上生成软件中断
#define __HAL_GPIO_EXTI_GENERATE_SWIT(__EXTI_LINE__) (EXTI->SWIER |= (__EXTI_LINE__))
```

## 使用示例

### 基本GPIO配置

```c
// 配置GPIO引脚为输出模式
GPIO_InitTypeDef GPIO_InitStruct = {0};

GPIO_InitStruct.Pin = GPIO_PIN_13;
GPIO_InitStruct.Mode = GPIO_MODE_OUTPUT_PP;
GPIO_InitStruct.Pull = GPIO_NOPULL;
GPIO_InitStruct.Speed = GPIO_SPEED_FREQ_LOW;

HAL_GPIO_Init(GPIOC, &GPIO_InitStruct);
```

### 外部中断配置

```c
// 配置GPIO引脚为外部中断模式
GPIO_InitTypeDef GPIO_InitStruct = {0};

GPIO_InitStruct.Pin = GPIO_PIN_0;
GPIO_InitStruct.Mode = GPIO_MODE_IT_RISING;
GPIO_InitStruct.Pull = GPIO_NOPULL;

HAL_GPIO_Init(GPIOA, &GPIO_InitStruct);

// 使能外部中断
HAL_NVIC_SetPriority(EXTI0_IRQn, 0, 0);
HAL_NVIC_EnableIRQ(EXTI0_IRQn);
```

### 外部中断处理

```c
// 外部中断处理函数
void EXTI0_IRQHandler(void)
{
  HAL_GPIO_EXTI_IRQHandler(GPIO_PIN_0);
}

// 外部中断回调函数
void HAL_GPIO_EXTI_Callback(uint16_t GPIO_Pin)
{
  if(GPIO_Pin == GPIO_PIN_0)
  {
    // 处理PA0引脚的中断
    HAL_GPIO_TogglePin(GPIOC, GPIO_PIN_13);
  }
}
```

## 注意事项

1. **引脚编号**: GPIO引脚编号从0到15，每个引脚都有对应的宏定义
2. **模式配置**: 确保选择正确的GPIO模式，特别是复用功能模式
3. **中断优先级**: 使用外部中断时，需要正确配置NVIC中断优先级
4. **时钟使能**: 使用GPIO端口前需要使能对应的GPIO时钟
5. **引脚锁定**: 使用HAL_GPIO_LockPin可以锁定引脚配置，防止意外修改
